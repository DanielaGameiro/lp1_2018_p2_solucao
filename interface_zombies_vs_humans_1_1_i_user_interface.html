Descrição detalhada

Interface que define de forma geral a funcionalidade de uma UI para o jogo Zombies vs Humans.

O jogo Zombie vs Humans suporta qualquer tipo de UI, gráfica ou texto, sendo suficiente que a classe que contém o código da UI implemente esta interface.

Definido na linha 22 do ficheiro IUserInterface.cs.

Documentação dos métodos

◆ Initialize()

void Initialize ( int  xDim,
int  yDim 
)

Inicializa e calcula todas as variáveis necessárias para desenhar a UI no ecrã, com base na dimensão do mundo.

Este método não deve apresentar nada no ecrã, apenas fazer os cálculos necessários para desenhar adequadamente a UI.

Parâmetros
xDimDimensão horizontal do mundo.
yDimDimensão vertical do mundo.

Implementado em ConsoleUserInterface.

Direção para a qual o agente se deve mover.

◆ RenderInfo()

void RenderInfo ( IDictionary< string, int >  info)

Apresenta e/ou atualiza o painel de informação contendo as estatísticas do jogo, por exemplo nº de turnos, nº de zombies, etc.

Parâmetros
infoDicionário contendo a informação/estatísticas a apresentar. As chaves são strings contendo o nome da informação, os valores são os valores inteiros que correspondem à informação a apresentar.
