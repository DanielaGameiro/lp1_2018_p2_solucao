Descrição detalhada

Struct que trata as opções inseridas pelo utilizador na linha de comandos, disponibilizando-as depois em propriedades.

Esta struct não pode ser instanciada diretamente uma vez que os seus construtores são privados. Para criar uma instância desta struct é necessário usar o método estático ParseArgs, que serve como "fábrica" de objetos deste tipo.
